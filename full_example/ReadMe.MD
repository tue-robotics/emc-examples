#### Terminal 1: build the project ####  
```bash
mkdir build
cd build
cmake ..
make
``` 

#### Terminal 2: start a roscore ####  
```bash
roscore
```

#### Terminal 3: start the simulator ####
```bash
mrc-sim
```

#### (optional) Terminal 4: visualize the sensor data  ####
```bash
mrc-viz
```

#### Terminal 1: run the program ####  
```bash
../bin/main
```
